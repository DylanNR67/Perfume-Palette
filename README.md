# 💎 Perfume Palette

A beautiful, highly trustworthy, and responsive luxury e-commerce website for **Perfume Palette** fragrance house.

Designed to bridge the gap between high-end luxury and modern accessibility, this site is built as an interactive Single Page Application (SPA) with smooth transitions, photorealistic assets, dynamic sillage/longevity metrics, and direct WhatsApp ordering.

---

## ✨ Features

1. **Homepage**:
   - Editorial luxury hero banner featuring tagline *"Find Your Perfect Scent Profile."*
   - Interactive **Scent Quiz Modal** that profiles user preferences and recommends the matching fragrance.
   - **Shop by Fragrance Family** category grid (Woody, Citrus, Floral, Oriental).
   - Click-and-drag **Best Sellers Carousel** showing top product cards.

2. **Product Page**:
   - Clean product gallery supporting detail notes.
   - Size/Medium selector that dynamically updates price fields.
   - Olfactory Notes Pyramid detailing Top, Heart, and Base notes.
   - Customer-rated **Sillage & Longevity sliders** with micro-animations on load.

3. **Scent Voices (Community & Reviews)**:
   - Stars rating overview and ratings breakdown distribution bar chart (histogram).
   - Sidebar filters to instantly sort cards by Fragrance Family, Longevity, or Occasion.
   - **Verified Buyer checks** and detailed attribute badges on review cards.
   - Interactive **Write a Review Form** to post comments in real-time.

4. **WhatsApp Ordering Flow**:
   - Action buttons on product cards, product details, and the cart drawer automatically open WhatsApp (`https://wa.link/x68he8`) to place orders directly with the seller.

5. **GitHub Integration**:
   - Linked repository shortcut in the navbar header and footer social cluster.

---

## 🎨 Design System

- **Color Palette**: Warm alabaster base (`#FAF8F5`), champagne cream containers (`#F3EFE9`), charcoal text (`#1A1A1A`), and muted metallic gold accents (`#C5A880`).
- **Typography**: Playfair Display (Serif) for headings; Inter (Sans-serif) for body and controls.
- **Interactivity**: Floating animation for hero bottles, soft blur drop shadows, glassmorphic drawers, and category zoom states.

## 🚀 Getting Started

### Local Development
To run this static website locally on your machine:

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:8080` in your browser.
